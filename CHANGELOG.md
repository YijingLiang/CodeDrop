# Changelog

## [1.0.0] - 2026-03-31

### Added
- Initial release of CodeDrop
- Menu bar SMS verification code monitoring
- Auto-detection and extraction of verification codes
- One-click copy to clipboard
- Desktop notifications
- Code history with timestamps
- Anthropic design theme (amber accent colors)

### Features
- Monitors for 3 minutes per activation
- Supports Chinese and English verification code formats
- Local processing only (no server, no cloud)
- Requires iPhone + Mac with same Apple ID and SMS Forwarding enabled
